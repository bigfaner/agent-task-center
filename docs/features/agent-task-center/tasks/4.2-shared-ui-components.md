# 4.2: Shared UI components

## Description

实现所有页面共用的基础 UI 组件，供后续页面任务复用。

**需实现的组件：**

**`AppHeader`** — 顶部导航栏
- Logo 文字 "Agent Task Center"
- 右侧 Upload 按钮，点击打开 `UploadDialog`

**`StatusBadge`** — Feature/Task 状态 badge
- 颜色映射：`prd`=blue, `design`=purple, `tasks`=yellow, `in-progress`=orange, `done`=green, `pending`=gray, `in_progress`=orange, `completed`=green, `blocked`=red

**`PriorityBadge`** — P0/P1/P2 badge
- P0: `bg-red-100 text-red-700`
- P1: `bg-orange-100 text-orange-700`
- P2: `bg-blue-100 text-blue-700`

**`CompletionRateBar`** — 进度条 + 百分比
- shadcn `Progress` 组件 + 右侧百分比文字（如 `75.0%`）

**`RelativeTime`** — 相对时间（`2h ago`, `1d ago`）
- 使用原生 `Intl.RelativeTimeFormat` 或简单手写

**`TableSkeleton`** — 表格骨架屏
- 接受 `rows` prop（默认 5），渲染灰色动画行

**`EmptyState`** — 空状态
- 接受 `title`、`description`、`action`（可选按钮）props

**`ErrorState`** — 错误状态
- 接受 `message`、`onRetry` props，显示红色提示 + 重试按钮

**`MarkdownRenderer`** — Markdown 渲染
- 封装 `react-markdown` + `remark-gfm`
- 添加 Tailwind `prose` 样式类

- `RelativeTime` 实现：计算 diff 秒数，按阈值显示 "Xs ago"/"Xm ago"/"Xh ago"/"Xd ago"
- 所有组件放在 `src/components/` 目录，每个组件一个文件
