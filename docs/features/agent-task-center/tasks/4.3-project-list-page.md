# 4.3: ProjectListPage

## Description

实现项目列表页 `/`，展示所有项目的汇总信息，支持名称搜索。

**组件层级：**
```
ProjectListPage
├── AppHeader (含 UploadButton)
├── SearchInput (debounce 300ms)
└── ProjectTable
    ├── ProjectTableRow (×N)
    └── TableSkeleton (loading)
```

**数据获取：**
```typescript
const { data, isLoading, error, refetch } = useQuery({
  queryKey: ['projects', search],
  queryFn: () => listProjects({ search, page: 1, pageSize: 20 }),
})
```

**搜索：** 前端 debounce 300ms 后更新 query key，触发重新请求（服务端过滤）。

**表格列：** 项目名称（链接）、Features 数、任务数、完成率（进度条）、更新时间（相对时间）。

**状态处理：**
- Loading → `TableSkeleton`（5 行）
- Error → `ErrorState` + 重试
- Empty（无数据）→ `EmptyState`（"暂无项目，点击上传开始"）
- Filtered Empty → `EmptyState`（"未找到匹配项目"）
