# 1.2: Config & DB connection setup

## Description

实现 `internal/config/` 配置加载和 `internal/db/` 数据库连接抽象层。

**Config 结构：**
```go
type Config struct {
    DBDriver  string // "sqlite" | "postgres"，默认 "sqlite"
    DBPath    string // SQLite 文件路径，默认 "./data/tasks.db"
    DBConnStr string // PostgreSQL DSN
    Port      int    // HTTP 端口，默认 8080
    StaticDir string // Web UI 静态文件目录
}
```

从环境变量读取：`DB_DRIVER`, `DB_PATH`, `DB_CONN_STR`, `PORT`, `STATIC_DIR`。

**DB 连接：**
- 根据 `DBDriver` 选择 SQLite（`modernc.org/sqlite`）或 PostgreSQL（`lib/pq`）
- 返回 `*sqlx.DB`，供后续 migration 和 query 使用
- SQLite 需设置 `_journal_mode=WAL` 和 `_foreign_keys=on`

## Implementation Notes

- config 包只做环境变量读取，不做业务逻辑
- db 包只做连接，不做 migration（migration 在 2.1 实现）
- 使用 `os.Getenv` + 默认值，不引入第三方配置库
