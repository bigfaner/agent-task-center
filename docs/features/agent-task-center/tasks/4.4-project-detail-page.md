# 4.4: ProjectDetailPage

## Description

实现项目详情页 `/projects/:id`，以 Tab 切换展示 Features 列表和 Proposals 列表。

**组件层级：**
```
ProjectDetailPage
├── PageHeader (← 返回 / 项目名 / UploadButton)
├── TabGroup
│   ├── Tab: Features (default)
│   └── Tab: Proposals
├── FeatureTable (Features tab)
└── ProposalTable (Proposals tab)
```

**数据获取：**
```typescript
const { data, isLoading, error } = useQuery({
  queryKey: ['project', id],
  queryFn: () => getProject(Number(id)),
})
```

**Features Tab 列：** Feature 名称（链接 → `/features/:id/tasks`）、状态 badge、完成率（进度条）、更新时间。

**Proposals Tab 列：** 提案标题（链接 → `/proposals/:id`）、关联 Features 数、创建时间。

**状态 badge 颜色：** 见 ui-design.md#status-badge-colors。

**状态处理：**
- Loading → 骨架屏（Tab + 表格行）
- Tab 内容为空 → "此项目暂无 Features/Proposals"
