# 1.1: Init monorepo scaffold

## Description

初始化 agent-task-center monorepo 目录结构，包含 server/（Go）和 web/（React + Vite）两个子项目，以及所有必要的配置文件。

目标结构：
```
agent-task-center/
├── server/
│   ├── cmd/server/main.go       # 占位入口
│   ├── internal/
│   │   ├── config/
│   │   ├── db/
│   │   ├── handler/
│   │   ├── model/
│   │   ├── parser/
│   │   └── service/
│   └── go.mod                   # module: agent-task-center/server
├── web/
│   ├── src/
│   │   ├── pages/
│   │   ├── components/
│   │   └── api/
│   ├── package.json
│   ├── vite.config.ts
│   ├── tailwind.config.ts
│   └── tsconfig.json
└── Makefile                     # build/run 快捷命令
```

## Acceptance Criteria

- [ ] `cd server && go build ./...` 编译成功（无业务代码，仅占位）
- [ ] `cd web && npm install && npm run build` 构建成功
- [ ] server go.mod 包含所有依赖：chi/v5, sqlx, modernc.org/sqlite, lib/pq, golang-migrate/migrate/v4
- [ ] web package.json 包含所有依赖：react, react-router-dom, @tanstack/react-query, react-markdown, remark-gfm, shadcn/ui, tailwindcss
- [ ] Makefile 包含 `make server`、`make web`、`make build` 目标

## Implementation Notes

- Go module 路径：`agent-task-center/server`
- Vite 配置需设置 `build.outDir: '../server/web/dist'`，供后续 embed.FS 使用
- shadcn/ui 初始化：`npx shadcn@latest init`，选择 New York style
- 所有 internal/ 子目录创建 `.gitkeep` 或占位 `.go` 文件保持目录结构
