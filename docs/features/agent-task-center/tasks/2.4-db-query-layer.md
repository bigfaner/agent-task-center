# 2.4: DB query layer (CRUD operations)

## Description

在 `internal/db/` 实现所有实体的数据库查询操作，供 Service 层调用。

**需实现的查询函数（按实体）：**

```go
// projects
func GetOrCreateProject(ctx, db, name) (*Project, error)
func ListProjects(ctx, db, search string, page, pageSize int) ([]Project, int, error)
func GetProject(ctx, db, id int64) (*Project, error)

// proposals
func UpsertProposal(ctx, db, projectID int64, input ProposalInput) (*Proposal, error)
func ListProposalsByProject(ctx, db, projectID int64) ([]Proposal, error)
func GetProposal(ctx, db, id int64) (*Proposal, error)

// features
func UpsertFeature(ctx, db, projectID int64, input FeatureInput) (*Feature, error)
func ListFeaturesByProject(ctx, db, projectID int64) ([]Feature, error)
func GetFeatureBySlug(ctx, db, projectID int64, slug string) (*Feature, error)

// tasks
func UpsertTask(ctx, db, featureID int64, input TaskInput) (*Task, error)
func ListTasksByFeature(ctx, db, featureID int64, filter TaskFilter) ([]Task, error)
func GetTask(ctx, db, id int64) (*Task, error)
func GetTaskByTaskID(ctx, db, featureID int64, taskID string) (*Task, error)
func ClaimTask(ctx, db, featureID int64, agentID string) (*Task, error)  // 乐观锁
func UpdateTaskStatus(ctx, db, id int64, agentID, status string) error

// execution_records
func InsertRecord(ctx, db, taskID int64, record ExecutionRecord) (*ExecutionRecord, error)
func ListRecordsByTask(ctx, db, taskID int64, page, pageSize int) ([]ExecutionRecord, int, error)
```

**ClaimTask 乐观锁实现：**
1. 查询 pending 且依赖已满足的任务，按 priority 排序（P0>P1>P2）
2. 取第一个候选，记录 (id, version)
3. `UPDATE tasks SET status='in_progress', claimed_by=?, version=version+1 WHERE id=? AND version=? AND status='pending'`
4. rows_affected=0 则取下一个候选，最多重试 3 次
5. 全部失败返回 `ErrNoAvailableTask`

## Acceptance Criteria

- [ ] 所有查询使用参数化查询（sqlx），无 SQL 拼接
- [ ] `UpsertTask` 不覆盖 `status`、`claimed_by`、`version` 字段
- [ ] `ClaimTask` 乐观锁逻辑正确，3 次重试后返回 `ErrNoAvailableTask`
- [ ] `UpdateTaskStatus` 验证 agentID 与 claimed_by 一致，不一致返回 `ErrUnauthorizedAgent`
- [ ] `go test ./internal/db/... -v` 使用 in-memory SQLite，覆盖率 ≥ 70%

## Implementation Notes

- 赖满足条件：`dependencies` JSON 数组中所有 task_id 对应的任务 status = 'completed'
- `ListTasksByFeature` 的 `TaskFilter` 过滤在 SQL WHERE 子句中实现（动态拼接条件）
- `sql.ErrNoRows` 统一包装为 `ErrNotFound`
- 所有函数接受 `context.Context` 作为第一个参数
